# ₹ Rupee Expense Tracker

A C++ expense tracker with an HTML dashboard to visualize spending.

## What it does
- Add expenses by category (Food, Transport, Entertainment, Shopping, Bills)
- Saves all data to expenses.json
- HTML dashboard shows total spent, category breakdown, and expense list

## How to run
g++ expense_tracker.cpp -o tracker

./tracker
Open index.html with VS Code Live Server to see the dashboard.

## Built with
- C++
- HTML/CSS/JavaScript
